Class & Id

Définir Class et id :

Tout ce qu'on a vu jusqu'à present implique par exemple que tous les paragraphes possèdent la même présentation.

Comment faire pour que cetains paragraphes seulement soiet ecrits d'une manière différente?

Pour répondre à la question, on peut utiliser deux (02) attributs spéciaux qui fonctionnent sure toutes les balises:
• Lattribut class et L'attribut id ces attributs permettent d'appliquer différents styles a des balises de meme type .

ID (identifiant) • Il fonctionne exactement de la même manière que « class », il ne peut etre utilis qu'une seule fois dans l'ordre.
• Il concerne les éléments uniques de structuration du document, comme les blocs principaux (logo, en-tête, colonne(s), pied de page...).
• Il est spécifié en CSS par un dièse (#) : #nom_id { propriété:valeur; }
class • l'attribut class peut se mettre sur n'importe quelle balise, aussi bien titre,paragraphe, image, etc…
• Permet d'appliqué un style à une classe definie de balises.
• Pour créer une classe, le sélecteur est constitué du nom choisi pour la classe pd du poit.
Exemple : .nom_class { propriété:valeur;}
