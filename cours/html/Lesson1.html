C'est quoi CSS?

  • CSS Signifie «feuille de style en cascade». Les feuilles de style en cascade sont utilisées pour formater la mise en page des pages Web
  • CSS permet aux développeurs Web de créer une apparence uniforme sur plusieurs pages d’un site Web
  • CSS facilite le changement de styles sur plusieurs pages à la fois

Pourquoi apprendre le CSS?

Hhaque page Web sans CSS serait en texte brut terne et . Avec CSS,vous Apprenez à styler et à organiser visuellement le code HTML avec CSS. vous pouvez ajouter des images de couleur et d'arrière-plan et modifier la mise en page de votre page - vos pages Web peuvent ressembler à des œuvres d'art!

Compétences à emporter:

Vous apprendrez de nombreux aspects du style des pages Web! Vous pourrez configurer la structure de fichier appropriée, modifier du texte et des couleurs et créer des présentations attrayantes. Grâce à ces compétences, vous pourrez personnaliser l'apparence de vos pages Web pour répondre à tous vos besoins!

Note sur les conditions préalables:

Nous vous recommandons de compléter Learn HTML avant d'apprendre CSS.


Après avoir passé toute une première partie du cours à ne travailler que sur le HTML, nous allons maintenant découvrir le CSS . Le CSS n'est pas plus compliqué que le HTML. Il vient le compléter pour vous aider à mettre en forme votre page web.

Quelques définitions:

Un sélecteur
détermine à quel élément un style doit être appliqué
Une propriété
détermine le style qui va être appliqué un élément
Une valeur
determine le comportement d'une propriété
