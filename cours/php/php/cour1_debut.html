Introduction

C'est quoi PHP?

PHP est un acronyme récursif pour "PHP: Hypertext Preprocessor".
PHP est un langage de script côté serveur intégré au HTML. Il est utilisé pour gérer le contenu dynamique, les bases de données, le suivi de session, voire même pour construire des sites de commerce électronique entiers.
Il est intégré à un certain nombre de bases de données populaires, notamment MySQL, PostgreSQL, Oracle, Sybase, Informix et Microsoft SQL Server.

Pourquoi le PHP?

PHP exécute des fonctions système, c’est-à-dire à partir de fichiers d’un système qu’il peut créer, ouvrir, lire, écrire et fermer.
PHP peut manipuler des formulaires, c'est-à-dire rassembler des données à partir de fichiers, enregistrer des données dans un fichier, envoyer par courrier électronique des données, renvoyer des données à l'utilisateur.
Vous ajoutez, supprimez, modifiez des éléments dans votre base de données via PHP.
Accéder aux variables de cookies et définir des cookies.
En utilisant PHP, vous pouvez restreindre l'accès des utilisateurs à certaines pages de votre site Web.
Il peut chiffrer des données.
